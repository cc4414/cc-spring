## 简介
cc-spring是一套基于spring的快速开发框架。它提供了一系列可插拔的模块，你可以利用它快速构建一个单体应用或者一个微服务应用。

cc-spring的各个模块分别整合了spring boot、spring cloud alibaba、mybatis plus、hutool等优秀的开源框架。
